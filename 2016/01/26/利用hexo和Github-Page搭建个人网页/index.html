利用hexo和Github Page搭建个人网页

发表于 2016-01-26

准备工作(prerequisite):

  1. Node.js的安装

    Windows: Node.js官网, 直接下载安装包安装.

  2. hexo 安装

    利用Node.js命令安装
    npm install hexo-cli -g

  3. Github Page Repository

    (1)Github 环境

    (2)Github Page Respository

    搭建流程

  4. hexo部署

    创建保存hexo网页工程的文件夹,并进入
    mkdir MyBlog cd MyBlog
    初始化部署hexo
    hexo init npm install

  5. hexo 配置

    打开_config.yml配置文件, 编辑deploy 部分:
    deploy: type: git repository: https://github.com/sylph520/sylph520.github.io.git branch: master
    并利用命令进行deploy type操作:
    npm install hexo-deployer-git --save

  6. hexo 清理,生成,部署

    hexo clean hexo generate hexo deploy
